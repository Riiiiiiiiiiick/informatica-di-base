Starting from the concepts contained in the book "The Stroke" written by Gerrit Noordzij, I've tried to apply with the code some effects to a letter path. In order to achieve these results, I've traced a simple line-path in illustrator. Then I've exported the .svg file and used the site PathToPoints to convert the svg line in points coordinates that I've later inserted in an array function. As the last thing I've coded the visual effect, that' made by the repetition of ellipses and a sort of halftone pattern created with the repetition of a line regulated by a for cycle.
